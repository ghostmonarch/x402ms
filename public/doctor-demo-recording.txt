# Monarch Doctor Demo
# Scenario: an agent is asked to build an x402 payment flow.
# Doctor should fail unsafe code, then pass after Monarch is installed.

## 1. Agent detects money-moving code and runs Doctor
$ node /Users/elijahpaul/x402ms/packages/x402/src/cli.js doctor

Preflight safety check for money-moving agent code.

# Changelog

## 0.1.0

Initial pre-release of the Monarch Shield agent payment safety kit.

- Added `monarch doctor` preflight gate.
- Added `doctor --ci` and `doctor --ci --strict`.
- Added local sandbox scenarios for `allow`, `caution`, `block`, and `route`.
- Added scanner for x402, paid MCP, pay-to wallet, and agent spend code.
- Added `checkPayment` and `checkBeforePayment` SDK exports.
- Added templates for x402 clients, paid MCP tools, agent wallets, and router integrations.
- Added real x402 integration example.
- Added agent-readable docs, `llms.txt`, `AGENTS.md`, and query-capture docs.
